# Muse 2.0.0

Muse 2.0.0 集中完成语音识别、文本注入、本地服务、模型安装、自动更新和发布供应链的安全加固。

## 本次更新

- 加固 ASR 会话生命周期、超时、断线重连和迟到回调隔离，降低旧会话覆盖新结果的风险。
- 改进文本注入与剪贴板事务，在焦点不可用或目标应用行为异常时提供可恢复回落。
- 收紧本地 SenseVoice、Qwen 服务的鉴权、进程身份、端口解析和输入边界。
- 为模型制品固定版本、大小和 SHA256，并采用 staging 安装、完整性验证及失败回滚。
- 建立 Cloud、Local 双制品链：Cloud 版不捆绑本地识别服务，Local 版包含冻结后的本地服务。
- 建立 Developer ID、Hardened Runtime、公证、Gatekeeper、双制品 SHA256 和远端资产复验门槛。

## 安装说明

- 支持 macOS 14 及以上版本。
- Cloud 与 Local 制品不能混用；请按是否需要本地识别服务选择对应 DMG。
- 覆盖安装后若无法向其他应用输入文字，请在“系统设置 → 隐私与安全性 → 辅助功能”中重新启用 Muse。
- 自动更新仍保持关闭；在双制品更新与回滚真机门槛全部完成前，请从 GitHub Release 手动下载安装。

## 发布验证

候选代码已通过 Debug/Release 构建、555 项 Swift 自动化测试（5 项按环境条件跳过）及仓库健康检查。正式 DMG 将在受保护的 GitHub 发布环境中完成 Developer ID 签名、公证、SHA256 和下载后复验。
